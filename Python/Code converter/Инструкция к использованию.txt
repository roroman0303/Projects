Инструкция к использованию:
1. Перенести в папку текстовый файл с программой
2. Открыть программу main
3. При необходимости изменить название текстового файла с программой
в строке : prog_text = CppWholeOpen("cpp_prog3.txt")
4. Запустить (F5)
5. При успешном выполнении программы в папке lastresults появится
текстовый файл (result+дата и время) с преобразованной программой
6. Кроме того, после выполнения вы можете найти в папке:
prep_prog.txt - файл с идентифицированными строками и комментариями
signed_prog.txt - файл с идентифицированным кодом программы
result_code.txt - файл с преобразованным кодом программы без анализа строк

Для корректной работы программы:
1. Соблюдайте принцип "В одной строке  - один оператор"
2. Фигурные скобки должны находиться в отдельной строке
и лишь обрамлять блоки кода
3. Соблюдайте отступы
4. Избегайте использования ссылок и адресов
5. Избегайте экранирования символов
При неправильном использовании программы возможны ошибки в работе
или пропуск части исходного кода.
